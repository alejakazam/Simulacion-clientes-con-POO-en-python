 Sistema de Gestión de Clientes en Python

Este proyecto consiste en una implementación básica de un sistema de gestión de clientes utilizando programación orientada a objetos en Python.
Permite crear instancias de clientes, almacenarlas y manipular sus datos de forma estructurada, simulando el comportamiento de una base de datos simple en memoria.

I. Características
Creación de clientes mediante clases
Almacenamiento de objetos en listas
Manejo de atributos como:
ID
Nombre
RUT
Teléfono
Dirección

II. Conceptos Aplicados
Programación Orientada a Objetos (POO)
Listas en Python
Encapsulamiento de datos
Simulación de base de datos relacional
